# Histórias de usuário
1. Como público geral, quero visualizar quais ações concretas o governo está tomando contra evasão escolar, para acompanhar o progresso do governo.
2. Como público geral, quero poder acompanhar a concretização de tarefas, para ver se estão sendo cumpridas.
3. Como público geral, quero buscar tarefas por palavras-chave, para encontrá-las mais faclmente.
4. Como público geral, quero poder visualizar o sistema em texto grande, para poder ler com clareza.
5. Como público geral, quero baixar dados de um indicador do sistema, para tê-los salvos em meu dispositivo.
6. Como secretária, quero inserir dados no sistema, para atualizar sua base de dados.
7. Como secretária, quero alterar o status de uma tarefa, para registrar seu progresso.
8. Como SEDEF, quero criar cadastro de usuário para nova secretária, para que ela possa gerenciar o sistema.
9. Como admin, quero deletar um cadastro de usuário do sistema, para impedir login indevido após desligamento do funcionário.
